Interactive REPL

Using the "py-repl" tag inside your HTML page, you are able to create interactive REPL boxes inside with full python capabilities.

In this example, weve created a basic Repl, as well as given it a custom id for CSS styling.

* For more complex package use within the Repl, youll need to add the desired packages inside a "py-config" tag within the head section of your HTML. This example Repl has the Pandas library included in the config.

We can test this by importing the libray and creating a sample dataframe with the following..

import pandas as pd

pd.util.testing.makeDataFrame()
