Custom .Py Imports

Got too much python code?

While this rarely happens IRL, if your page gets bogged down with code, break it up!

Take all that python code and move it to a seperate ".py" file, then load it into the html page using the "src" property of your py-script tag.

"py-script src="myCustomScript.py"

In the example below, the button is linked to a seperate ".py" file with a random number generator. Try it out!
